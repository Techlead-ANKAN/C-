Objects - These are the entities in the real world.

Class - These are like blueprints of these entities.

Syntax for creating a class: -

class Teacher {
    // attributes
};


Access Modifiers : -

1) "private" (by default) - data and methods are accessible inside class

2) "public" - data and methods are accessible to everyone

3) "protected" - data and methods are accessible inside class and to its iderived class.
